---

### Features

- Fully structured **Amazon-like layout** (navbar, secondary panel, hero section, shop section, footer)
- Multiple **product category boxes** with images and titles
- Functional search bar (visual only, no live API)
- Hover effects on navbar and boxes
- Styled footer with multiple panels and links
- Clean, readable **HTML & CSS** structure

---

### Project Structure / Components

- **Navbar**: Main logo, location, search bar, account, returns, and cart  
- **Panel**: Secondary menu with category options  
- **Hero Section**: Banner image section  
- **Shop Section**: Grid of product category boxes with images  
- **Footer**: 4 panels with links, language selector, and copyright  
- **CSS Styling**: Flexbox layout, hover effects, box spacing, fonts
